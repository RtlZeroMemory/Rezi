# @rezi-ui/ink-compat

Fresh-start Ink compatibility adapter package for Rezi.

This package targets mainstream Ink API compatibility first and is locked to:
- Ink `6.7.0`
- Ink commit `135cb23ae3b7ca94918b1cd913682f6356f12c5c`

See `docs/ink-compat/` for lock, invariants, architecture, parity matrix, and missing capabilities.
